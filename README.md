# Callit — The Prediction Market for the Future

Callit is a social prediction platform where people make **calls about the future**.

Users can post predictions about sports, crypto, technology, politics, culture and more. Others can agree, disagree, or follow the call as it evolves.

Think of it as a place where **opinions meet foresight**.

---

## Core Idea

The world runs on predictions.

Callit gives anyone the ability to say:

- *“This will happen.”*
- *“I call it.”*

And let the world decide.

---

## Features

• Create predictions ("calls") about real world events  
• Explore predictions across categories  
• Vote **Yes / No** on outcomes  
• Track trending opinions  
• Browse leagues and competitions in sports  
• Real-time topics powered by Supabase  

---

## Categories

Callit currently supports predictions across:

- Sports
- Crypto
- Technology
- Politics
- Entertainment
- Global Trends

Sports leagues include:

- English Premier League
- La Liga
- NBA
- WNBA
- Esports

More categories will be added as the platform evolves.
